Arquitetura

O Fluxo dos Dados

Entenda o caminho que a informação percorre, desde a prefeitura até a tela do seu computador. Uma arquitetura simples, sem servidores caros, focada em performance.

📊

1. Dados Brutos

Planilha CSV gigante da SEFIN (Prefeitura). Contém todos os registros de imóveis desde 1996, mas com falhas de formatação e sujeira.

⚙️

2. Limpeza (ETL)

Um script em Python analisa, limpa erros, calcula novos valores (como valor do m²) e gera um arquivo leve e perfeito.

💻

3. Dashboard

Aplicativo Web construído em React. Ele lê o arquivo limpo instantaneamente e cria os gráficos e mapas que você interage.

Parceria com IA

Como a Inteligência Artificial Ajudou?

Para os membros da equipe que não são programadores: nós usamos ferramentas avançadas de IA para nos ajudar a construir este projeto muito mais rápido e com melhor qualidade.

🚀

Claude Code & Antigravity

Utilizamos assistentes de programação baseados em IA (Agentes Autônomos) que funcionam como um co-piloto para o desenvolvedor.

  • Velocidade: Escrevem código repetitivo em segundos.
  • Resolução de Problemas: Ajudam a encontrar erros difíceis de visualizar a olho nu.
  • Refatoração: Melhoram a estrutura do código automaticamente.
🐍

Criação do Script Python

A limpeza de dados é complexa. A IA foi fundamental para construir a lógica pesada de transformação (ETL).

  • Lidou com caracteres quebrados no CSV da prefeitura.
  • Criou algoritmos de cálculo para a idade dos imóveis e faixas de preço.
  • Garantiu que não perdêssemos nenhum dado no processo.
🎨

Design e UI/UX

A criação visual também teve suporte inteligente para garantir que a interface fosse amigável e moderna.

  • Sugeriu animações fluidas (como as entradas de tela).
  • Ajudou na componentização dos gráficos complexos do Recharts.
  • Auxiliou a criar a lógica que filtra todos os dados instantaneamente.

Histórico

A Jornada do Projeto

Passo 1

Análise dos Dados Brutos

Pegamos a base de dados do Portal da Prefeitura (SEFIN). Identificamos que o formato era complexo e precisaria de uma limpeza profunda antes de poder virar gráficos.

Passo 2

Construção do ETL com Python + IA

Com a ajuda da IA, construímos scripts em Python que corrigem acentos, removem duplicatas e calculam métricas valiosas como o valor do m² de cada transação.

Passo 3

Design e Interface Web

Criamos um Design System usando as cores oficiais da prefeitura. Implementamos a interface em React focando na melhor experiência de usuário possível (UX).

Passo 4

Visualizações e Filtros

Adicionamos os gráficos (Recharts) e o mapa interativo (Leaflet). Criamos o sistema de filtros global, onde clicar em um bairro atualiza todos os outros dados em tempo real.
